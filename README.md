# Block Breaker Game in Assembly

## Overview
This project implements a simple block breaker game using Assembly language.

The game demonstrates low-level programming concepts including memory handling, control flow, and real-time input/output.

---

## Features
- Interactive gameplay
- Block collision detection
- Score tracking
- Real-time updates
